# Regras e Referência Rápida - Starfinder 2e

Este documento contém um resumo das condições e ações principais do sistema para consulta rápida através do Stylish Action HUD.

## Condições (Conditions)

**Legenda de Ações:**
*   <span style="font-family: Pathfinder2eActions">1</span> Uma Ação | <span style="font-family: Pathfinder2eActions">2</span> Duas Ações | <span style="font-family: Pathfinder2eActions">3</span> Três Ações
*   <span style="font-family: Pathfinder2eActions">R</span> Reação | <span style="font-family: Pathfinder2eActions">F</span> Ação Livre
*   **(E)** Atividade de Exploração
*   **(D)** Atividade de Downtime
*   **(T)** Exige Proficiência Treinado

| Condição | Descrição |
| :--- | :--- |
| **Blinded** | Visão impedida; terreno normal vira difícil; falha automática em Percepção visual; imune a efeitos visuais. |
| **Broken** | Objeto inoperante; armaduras dão penalidade de status na CA. |
| **Clumsy** | Penalidade de status em Destreza (CA, Reflexos, ataques à distância). |
| **Confused** | Off-guard; sem aliados; sem reações; ataca alvos aleatórios. |
| **Controlled** | Suas ações são ditadas por quem te controla. |
| **Dazzled** | Criaturas e objetos são considerados ocultos (concealed) para você. |
| **Deafened** | Audição impedida; falha automática em Percepção auditiva; penalidade em iniciativa sonora. |
| **Drained** | Penalidade de status em Constituição; reduz PV atuais e máximos conforme o nível. |
| **Doomed** | Reduz o limite de Dying necessário para a morte instantânea. |
| **Encumbered** | Condição Clumsy 1 e penalidade de -10 pés em todas as Velocidades. |
| **Enfeebled** | Penalidade de status em Força (ataques corpo a corpo e dano). |
| **Fascinated** | Penalidade em Percepção e perícias; não pode usar ações de concentração (exceto para o alvo). |
| **Fleeing** | Deve gastar todas as ações fugindo da fonte da condição. |
| **Frightened** | Penalidade de status em todos os testes e DCs; diminui 1 por turno. |
| **Glitching** | Exige teste fixo para agir ou usar itens; falhas impõem penalidades ou atordoamento. |
| **Grabbed** | Imobilizado e off-guard; exige teste fixo para ações de manipulação. |
| **Immobilized** | Não pode realizar ações com o traço Move. |
| **Off-Guard** | Penalidade de circunstância de -2 na Classe de Armadura (CA). |
| **Paralyzed** | Off-guard; apenas ações puramente mentais (ex: Recall Knowledge). |
| **Persistent Damage** | Sofre dano no fim do turno; teste fixo para encerrar. |
| **Petrified** | Não pode agir ou sentir; Bulk dobrado. |
| **Prone** | Off-guard; penalidade em ataques; movimentos limitados a Crawl e Stand. |
| **Quickened** | Ganha 1 ação adicional por turno. |
| **Restrained** | Grabbed aprimorado; imobilizado; off-guard; apenas ações Escape ou Force Open. |
| **Sickened** | Penalidade em testes e DCs; não pode ingerir itens; ação Retch reduz valor. |
| **Slowed** | Reduz o número de ações recuperadas no início do turno. |
| **Stunned** | Não pode agir; perde ações no início do turno conforme o valor. |
| **Stupefied** | Penalidade em Inteligência, Sabedoria e Carisma; risco de falha em magias. |
| **Suppressed** | Penalidade de circunstância em ataques e penalidade de status na Velocidade. |
| **Untethered** | Gravidade zero; Clumsy 1, off-guard; flutua 5 pés no fim do turno. |
