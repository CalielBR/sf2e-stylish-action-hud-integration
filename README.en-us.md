## ✨ Key Features

- 🔗 **Tactical Optimization**: Designed for high-tech combat (SF2e / PF2e engine).  
- ⚡ **Automated HUD**: Automatically populates actions, skills, spells, and equipment.  
- 🧼 **Clean Interface**: Organized design to reduce visual clutter.  
- 📏 **Adjusted Submenus**: Optimized width for long ability names.  
- 🚀 **Plug and Play**: Works immediately with no configuration required.  
- 🧩 **Compatibility**: Fully compatible with Foundry VTT V12 and V13.  

---

## 📦 Requirements

This module requires:

- **System**: Starfinder 2nd Edition (v0.0.11)  
- **Module**: [Stylish Action HUD](https://github.com/glitchsmith/stylish-action-hud)

---

## 🚀 Installation

1. Open Foundry VTT  
2. Go to **Add-on Modules**  
3. Click **Install Module**  
4. Paste the Manifest URL:

https://github.com/CalielBR/sf2e-stylish-action-hud/releases/latest/download/module.json

---

## 🧠 How It Works

This module adapts Stylish Action HUD for Starfinder 2e by organizing:

- Actions  
- Spells  
- Skills  
- Equipment  

👉 Everything is optimized for clarity and efficient screen usage.
