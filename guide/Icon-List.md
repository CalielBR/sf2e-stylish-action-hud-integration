# Material Symbols Icons - Starfinder 2e HUD

This list contains the ligatures (icon names) used in the **Stylish Bridge - SF2e** module. Since we are using a single font file (`.woff2`), you do not need to download individual files, but you can use this list for aesthetic reference.

## 1. Main Attributes (HUD Bar)

- `favorite` : Hit Points (HP) - (Material Symbols: favorite)
- `bolt` : Stamina Points (SP) - (Material Symbols: bolt)
- `shield` : Armor Class (AC) - (Material Symbols: shield)
- `psychology` : Resolve Points (RP) - (Material Symbols: psychology)
