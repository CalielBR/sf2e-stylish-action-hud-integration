## 🇬🇧 English

### 🌟 Key Features

- **Starfinder 2e Themed**: Visual style updated with SF2e purple colors and high-tech icons.
- **Enhanced Attribute Tracking**: Dedicated bars for HP and Stamina (SP), plus badges for AC and Resolve Points (RP).
- **Automated HUD Organization**:
  - **Encounter**: Attacks, Movement, Defense, and Skill Actions.
  - **Exploration & Downtime**: Organized activities for travel and rest periods.
- **Smart MAP Integration**: Built-in buttons for Multiple Attack Penalty on Strikes and relevant Skill Actions (Trip, Grapple, etc.).
- **System Specific Utilities**:
  - Custom Recovery Check logic.
  - Interactive Dying and Wounded trackers.
  - Integrated Recall Knowledge dialog for SF2e skills.
- **Inventory Management**: Specialized categories for Augmentations, Upgrades, Fusions, and Ammunition.
- **Stamina Logic**: Automatically handles damage absorption where Stamina is reduced before HP.

### 🛠️ Requirements

- **System**: [Starfinder 2nd Edition](https://foundryvtt.com/packages/sf2e) (v0.2.0+)
- **Module**: [Stylish Action HUD](https://foundryvtt.com/packages/stylish-action-hud)

### 🚀 Installation

1.  In Foundry VTT, go to the **Add-on Modules** tab in the Configuration and Setup screen.
2.  Click **Install Module**.
3.  In the "Manifest URL" field, paste the following link:
    ```
    https://raw.githubusercontent.com/MikeBarbosa/starfinder2e-stylish-action-hud/main/module.json
    ```
4.  Click **Install** and wait for the installation to complete.
5.  Activate this module and all required dependencies in your game world's module settings.

## 🇧🇷 Português

### 🌟 Principais Funcionalidades

- **Temática Starfinder 2e**: Visual atualizado com as cores roxas do SF2e e ícones tecnológicos.
- **Monitoramento de Atributos**: Barras dedicadas para HP e Stamina (SP), além de badges para CA e Pontos de Determinação (RP).
- **Organização Automática**:
  - **Encounter**: Ataques, Movimentação, Defesa e Ações de Perícia.
  - **Exploration & Downtime**: Atividades organizadas para viagens e períodos de descanso.
- **Integração de MAP**: Botões integrados para Penalidade de Ataques Múltiplos em Ataques e Ações de Perícia (Tropeçar, Agarrar, etc).
- **Utilitários do Sistema**:
  - Lógica personalizada para Teste de Recuperação (Recovery Check).
  - Marcadores interativos de Moribundo (Dying) e Ferido (Wounded).
  - Diálogo de Recordar Conhecimento (Recall Knowledge) integrado.
- **Gestão de Inventário**: Categorias especializadas para Melhorias (Upgrades), Fusões, Aumentos (Augmentations) e Munição.
- **Lógica de Stamina**: Gerencia automaticamente a absorção de dano onde a Stamina é reduzida antes do HP.

### 🛠️ Requisitos

- **Sistema**: Starfinder 2nd Edition (v0.2.0+)
- **Módulo**: Stylish Action HUD

### 🚀 Instalação

1.  No Foundry VTT, vá para a aba **Add-on Modules** na tela de Configuração.
2.  Clique em **Install Module**.
3.  No campo "Manifest URL", cole o seguinte link:
    ```
    https://raw.githubusercontent.com/MikeBarbosa/starfinder2e-stylish-action-hud/main/module.json
    ```
4.  Clique em **Install** e aguarde a conclusão da instalação.
5.  Ative este módulo e todas as suas dependências nas configurações de módulos do seu mundo de jogo.
